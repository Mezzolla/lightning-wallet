# Lightning Wallet

A modern, user-friendly Bitcoin Lightning Network wallet built with Next.js and React.

## Features

- **Bitcoin and Lightning Support**: Manage both on-chain Bitcoin and Lightning Network transactions
- **Lightning Channels**: Open, manage, and monitor Lightning Network channels
- **Quick Send/Receive**: Easy interface for sending and receiving payments
- **Contact Management**: Save and organize payment recipients
- **Transaction History**: Track all your Bitcoin and Lightning transactions
- **Security-focused**: Built with security best practices
- **Modern UI**: Clean, responsive interface built with Tailwind CSS and Radix UI
- **Self-sovereign**: Control your own keys and funds

## Technology Stack

- **Framework**: Next.js 15
- **UI**: React 19, Tailwind CSS, Radix UI components
- **State Management**: React Context API
- **Forms**: React Hook Form with Zod validation
- **Charts & Visualizations**: Recharts
- **Development**: TypeScript, ESLint

## Getting Started

### Prerequisites

- Node.js 18+ and npm/pnpm

### Installation

1. Clone the repository

   ```
   git clone https://github.com/yourusername/lightning-wallet.git
   cd lightning-wallet
   ```

2. Install dependencies

   ```
   npm install
   # or
   pnpm install
   ```

3. Start the development server

   ```
   npm run dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

1. **Onboarding**: Complete the initial setup process to create or restore a wallet
2. **Dashboard**: View your balance, recent transactions, and Lightning channels
3. **Send**: Send Bitcoin or Lightning payments
4. **Receive**: Generate invoices or addresses to receive payments
5. **Lightning**: Manage Lightning Network channels and peers
6. **Settings**: Configure wallet preferences and security settings
